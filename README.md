# Emergency Callouts v1.0 FivePD
A collection of callouts my friend and I made for FivePD and ran on New Night and the beginning of True Night. I was one of the 2 primary developers during that period that allowed us to become one of the top FivePD servers to exist at the time.
Features : A list of callouts including the possibility of false 911s, wandering suspects, silver alert, and more~
some have 1000+ set locations they can spawn at, others use random modifiers. Unfortunately the last release of FivePD broke a lot of functionality within APIs including text
